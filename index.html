Sazın qaynaq kodu .saz fayl uzantısı ilə saxlanılır. Daha sonra siz

NÜMUNƏLƏR

Aşağıdakı nümunə kodlarını hər hansı bir fayla UTF-8 kodlaşdırma ilə yazdıqdan sonra saz [fayl-adı] komandası ilə proqramı başlada bilərsiniz.

Yox əgər fayla yazmaq istəmirsinizsə saz komandası ilə saz shell‘ni başladaraq program kodunu shell‘ə yazmağınız kifayətdir.

SALAM DÜNYA

yaz("salam dünya");


DƏYİŞƏNLƏR

Saz dilində 3 müxtəlif dəyişən tipi mövcuddur. mətn, ədəd və məntiqi

// mətn
ad = "Məlikməmməd";

// ədəd
boy = 1.75;

// məntiqi
a = doğru;
b = yanlış;

Əvvəlcədən təyin olunmuş dəyişənləri yenidən təyin etmək mümkündür. Bu halda köhnə məlumat silinəcək və yenisi ilə əvəz olunacaq.

boy = "Oğuz";
boy = 1.75;

Təyin olunmamış dəyişənlər boş dəyərinə məxsusdur. Dəyişənə boş mənimsətmək onu silir.

ad = "Məlikməmməd";
ad = boş;

yerli dəyişənlər

Yerli dəyişənlər sadəcə təyin olunduğu sahəyə məxsusdurlar. Və bu sahə sona çatanda silinirlər.

{
  yerli alma = 10;
  yaz(alma);  // 10
}
yaz(alma);  // boş

Bu tip dəyişənlər kölgələmə xüsusiyyətinə malikdirlər. Yəni o adda qabaqcadan təyin olunmuş dəyişənləri kölgələyirlər.

alma = 5;
{
  yerli alma = 10;
  yaz(alma); // 10
}
yaz(alma); // 5


ƏGƏR-BAŞQA

Verilmiş şərt doğru olduğu təqdirdə bəyanatın əsas gövdəsini yerinə yetirir. Əks halda isə başqa gövdəsini.

icazəVar = doğru;

əgər(icazəVar) {
  yaz("icazə var");
} başqa {
  yaz("icazə yoxdur");
}

Sadəcə əsas gövdədən ibarət əgər bəyanatı da mümkündür.

icazəVar = doğru;

əgər(icazəVar) {
  yaz("icazə var");
}

əgər bəyanatının başqa gövdəsinə yeni əgər bəyanatı calayaraq daha mürəkkəb bəyanat yaratmaq mümkündür.

a = 7;

əgər(a > 0) {
  yaz("a ədədi müsbətdir")
} başqa əgər(a < 0) {
  yaz("a ədədi mənfidir")
} başqa {
  yaz("a ədədi nə müsbət nə də mənfidir")
}

// nəticə: a ədədi müsbətdir


NƏQƏDƏRKİ

Verilmiş şərt doğru olduğu müddətcə bəyanatın gövdəsi yerinə yetirilir.

a = 0;

nəqədərki(a < 10)
{
  yaz(a + "\n");
  a++;
}
